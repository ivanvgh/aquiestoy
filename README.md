# Aquiestoy 🔧

Marketplace de servicios on-demand en Arequipa, Perú.

Conectamos clientes con técnicos independientes verificados (gasfitería, electricidad, cerrajería, pintura y más). Cotizaciones transparentes por hora, técnicos con antecedentes penales verificados, y un sistema de reputación que incentiva la calidad.
